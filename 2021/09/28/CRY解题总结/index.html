变异凯撒（变异的ASCII）

这题看着挺懵的，用凯撒解密，当然是解不出来。

刚学密码学，只好百度了。。

afZ_ ASCII码分别是97,102,90,95;而flag的ASCII码则是102 108 97 103，脑洞打开：97+5=102；102+6=108；90+7=97；95+8=103，以此类推，利用C语言来算

#include<stdio.h> int main() { int i; char a[30]=“afZ_r9VYfScOeO_UL^RWUc”; for(i=0;a[i];i++) { a[i]=a[i]+i+5; printf("%c",a[i]); } return 0; }
